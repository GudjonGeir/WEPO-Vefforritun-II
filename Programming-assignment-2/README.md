## Chatter

#### Requirements
* Node.js
* Bower
* Grunt

#### Installation
Run the following commands:
```bash
bin/install.bat
```
Then
```bash
grunt
```

#### Running the application
Start the server:
```bash
bin/start-server.bat
```
Start the client:
```bash
bin/start-client.bat
```
Load the app at [http://localhost:8000/](http://localhost:8000/)
